# CSP451 CheckPoint 1

## Project Overview
This lab assignment introduces the fundamentals of Git and version control by guiding students through the creation and management of a Git repository. Students perform essential Git operations such as initializing a repository, committing changes, creating branches, and merging content, while also collaborating through GitHub.

## Key Concepts
- **Version Control:** Track changes to files, collaborate, view history, and revert changes.
- **Git:** Distributed version control system with:
  - Branching and merging
  - Staging and committing changes
  - Viewing history with `git log`
  - Collaboration through GitHub
